## Tech Stack
- Next.js (create-next-app)
- Typescript
- React Query
- Tailwind
- Vercel Deployment
- Atomic Design Pattern
- Testing-library + Jest
- Eslint + Prettier

## Features
- Support dark/light mode by color scheme system
- Responsive Design
- Live Price Update
- Atomic Design Pattern
